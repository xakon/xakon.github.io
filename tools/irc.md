IRC
===

Using IRC and some clients.

Usage
-----

### Exit IRC

```
/quit
```

### Register nickname

```
/msg NickServ REGISTER password xakon@yahoo.com
```

### Identify

```
/msg NickServ IDENTIFY nick nickserv_password
```

### List Channels

```
/list
```

### Join Channel

If a channel doesn't exist, the `/join` command will create it.

```
/join #haskell
```

### Leave Channel

```
/leave
```
