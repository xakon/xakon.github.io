Mac OS X
========

Some useful tools of Mac OS X that have similar Linux counterparts,
but different names:

 - `pbcopy`, `pbpaste`
 - `open`
 - `mdfind`
 - `screencapture`
 - `launchctl`
 - `say`
 - `diskutil`
 - `brew`
